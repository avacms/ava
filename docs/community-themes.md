## Submit Your Theme

Built a theme you'd like to share? We'd love to feature it here!

**What to include:**
- Theme name and a short description
- Link to the GitHub repo (or other download source)
- A screenshot or preview link
- Which Ava version you tested it with

**How to submit:**
- [Open an issue on GitHub](https://github.com/ava-cms/ava/issues) with the label "community-theme"
- Or share it in [Discord](https://discord.gg/Z7bF9YeK)
